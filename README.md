# Seabird

[![Build Status](https://travis-ci.org/belak/go-seabird.svg?branch=master)](https://travis-ci.org/belak/go-seabird)

## Requirements

* go >= 1.12
* gcc

# Running

```
SEABIRD_CONFIG=$HOME/config.toml go run cmd/seabird/main.go
```
